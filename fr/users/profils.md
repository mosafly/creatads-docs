# Profils d'audience

Les profils sont des personas IA qui représentent vos segments cibles. Chaque profil oriente la génération vers un message et un style visuel adaptés à ce public.

---

## Ce qu'est un profil

Un profil contient trois éléments :

| Champ | Description | Exemple |
|---|---|---|
| **Persona** | Qui est cette personne | "Femme 30-45 ans, CSP+, urbaine, sensible à l'authenticité" |
| **Pain point** | Sa douleur ou son frein principal | "Peur d'offrir un cadeau impersonnel ou décevant" |
| **Angle** | L'axe de message à utiliser | "Approche 'Transmission' : valoriser le savoir-faire artisanal comme preuve de valeur" |

L'IA utilise ces trois éléments pour adapter la composition, les couleurs dominantes, et les textes générés.

---

## Générer des profils

La méthode la plus rapide : laissez l'IA générer **10 profils** depuis une description de votre marque.

1. Allez dans **Brand Setup → Profils**
2. Cliquez sur **Générer des profils**
3. Entrez une description de votre marque (plus elle est riche, meilleurs sont les profils)
4. Sélectionnez la langue de sortie

**Exemple de description efficace :**
```
Marque de chocolat artisanal française, positionnement premium, 
marché du cadeau, ciblant les 30-55 ans cherchant des cadeaux 
de qualité pour des occasions spéciales. Différenciateur : 
bean-to-bar, origines uniques, ateliers parisiens.
```

La génération prend **30–60 secondes** et produit 10 profils distincts, immédiatement disponibles dans vos campagnes.

---

## Choisir les bons profils

Après génération, lisez chaque profil et notez les 2–3 les plus pertinents pour votre prochaine campagne.

**Critères de sélection :**
- Le persona correspond à votre cible prioritaire
- Le pain point reflète un vrai frein à l'achat
- L'angle est cohérent avec votre message commercial actuel

---

## Utiliser les profils en campagne

Dans le wizard de création de campagne (Étape 2), sélectionnez jusqu'à 12 profils. Chaque profil génère ses propres visuels — vous obtenez ainsi plusieurs angles créatifs en un seul batch.

→ Voir [Campagnes](./campagnes.md)

---

## Gérer vos profils

Les profils sont liés à l'espace client actif. Vous pouvez :
- Consulter le détail complet (persona, pain point, angle)
- Filtrer vos créatifs existants par profil
- Ajouter des profils manuellement si vous avez déjà des personas définis

> Les profils sont partagés entre toutes vos campagnes pour ce client. Si vos cibles évoluent, régénérez pour obtenir des profils actualisés.
