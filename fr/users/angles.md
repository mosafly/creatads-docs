# Angles créatifs

Les **angles** sont la pièce maîtresse de CreatAds. Ils remplacent l'ancien concept de "profil d'audience" pour s'aligner sur la réalité de Meta Andromeda (Advantage+) : aujourd'hui, ce n'est plus le ciblage qui décide qui voit votre pub — c'est la diversité créative qui décide ce qui convertit.

Votre travail n'est plus de définir 10 personas démographiques, mais de fournir à l'algorithme **10 angles créatifs distincts** pour qu'il apprenne, format par format, ce qui marche.

---

## Ce qu'est un angle

Un angle créatif a quatre dimensions :

| Champ | Description | Exemple |
|---|---|---|
| **Nom** | Étiquette courte interne (5 mots max) | "Pain – cernes matin" |
| **Hook** | La phrase d'accroche (8–15 mots) | "Vos cernes vous trahissent dès la première réunion du matin." |
| **Niveau de conscience** | Où en est l'audience | `unaware` / `problem_aware` / `solution_aware` / `product_aware` / `most_aware` |
| **Style visuel** | Le look de la créa | `ugc` / `studio` / `lifestyle` / `text_heavy` / `before_after` / `demo` |
| **Direction copy** | Ton + bénéfice clé + suggestion CTA | "Ton confident, urgent. Lead avec -30%. CTA : Découvrir." |

L'IA injecte ces quatre dimensions dans le prompt envoyé au modèle d'image et dans la copy générée — chaque angle donne donc une créa visuellement et message-distincte.

---

## Pourquoi cette structure

**Niveau de conscience** (Eugene Schwartz) — détermine à qui parle la pub :
- `unaware` → curiosité, reveal de problème
- `problem_aware` → on appuie sur la douleur
- `solution_aware` → comparaison, why-us
- `product_aware` → offre, urgence
- `most_aware` → social proof, FOMO, fidélisation

**Style visuel** — détermine le format. Meta favorise les comptes qui mixent UGC, studio, lifestyle et démo plutôt qu'un seul look répété.

**Hook** — c'est la promesse de la première seconde. Sans hook, l'algorithme ne peut pas optimiser le CTR.

**Direction copy** — guide la génération des textes (headlines, body, description) au moment de la préparation de campagne.

## Générer 10 angles avec l'IA

La méthode recommandée : laissez l'IA générer **10 angles diversifiés** depuis votre description de marque.

1. Allez dans **Brand Setup → Angles créatifs**
2. Renseignez le résumé de recherche dans le panneau "Recherche / brief"
3. Cliquez sur **Générer**
4. Sélectionnez la langue de sortie

L'IA produit **10 angles distribués selon une règle Andromeda-native** :
- 2× unaware
- 3× problem_aware
- 2× solution_aware
- 2× product_aware
- 1× most_aware

Avec au moins **4 styles visuels différents** parmi les 6 disponibles.

**Exemple de description efficace :**
```
Marque de chocolat artisanal français, positionnement premium, marché du cadeau,
30-55 ans cherchant des cadeaux de qualité. Différenciateur : bean-to-bar,
origines uniques, ateliers parisiens.
```

Génération en **30–60 secondes**, immédiatement utilisable dans vos campagnes.

---

## Créer un angle manuellement

Si vous avez une idée précise :

1. Dans **Brand Setup → Angles**, cliquez sur **Ajouter**
2. Renseignez les 5 champs (nom, hook, niveau de conscience, style visuel, direction copy)
3. Validez

L'angle apparaît dans la liste, utilisable immédiatement.

---

## Utiliser les angles en campagne

Dans le wizard de création de campagne (étape 2), sélectionnez jusqu'à 12 angles. Chaque angle :
- génère ses propres visuels (selon son style visuel)
- reçoit sa propre copy (selon sa direction copy)
- crée son propre adset dans Meta Ads (selon son niveau de conscience → top/mid/bottom funnel)

→ Voir [Campagnes](./campagnes.md)

---

## Gérer vos angles

Les angles sont liés à l'espace client actif. Vous pouvez :
- Consulter le détail (nom, hook, niveau de conscience, style visuel, direction copy)
- Filtrer vos créatives existantes par angle
- Régénérer le pool de 10 angles si votre positionnement évolue

> Les angles sont partagés entre toutes vos campagnes pour ce client. Si votre offre change, régénérez pour obtenir un nouveau pool aligné.
